# 🕒 Smart Digital Clock using Python

## 📌 Project Overview:

The Smart Digital Clock is a Python GUI application developed using the Tkinter library. It displays the current time, date, day, and a dynamic greeting message based on the time of the day. The application updates automatically every second, providing a real-time digital clock experience with a modern user interface.
This project was completed as part of my Python Internship at InternPe.

## 🚀 Features:

* Real-time Digital Clock

* Current Date Display

* Current Day Display

* Dynamic Greeting Message
  * Good Morning

  * Good Afternoon

  * Good Evening

* Modern GUI Design

* Automatic Time Update Every Second

* User-Friendly Interface

## 🛠️ Technologies Used:

* Python

* Tkinter

* Time Module

* Datetime Module
